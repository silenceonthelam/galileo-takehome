## What is It?
This project creates a master view for a user to add providers (doctors).

## How to View the Example

1. Install dependencies
``` npm i ```

2. Enter the directory
``` cd galileo takehome ```

3. Run the project
``` npm run dev ```

4. View in browser
Go to ```localhost:1234``` in your browser.

5. Enjoy?
This step is optional.
